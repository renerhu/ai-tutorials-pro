Midjourney Tutorial: From Beginner to Pro in 30 Minutes

📅 May 29, 2026 • 📖 12 min read • 🏷️ Tutorials

Midjourney has become the go-to AI image generator for designers, marketers, and creators. But there's a difference between typing a random prompt and actually controlling what Midjourney produces. This guide takes you from zero to confident in one sitting.

1. Getting Started with Midjourney

Midjourney (currently v6.2 as of mid-2026) operates entirely through Discord. Here's what you need:

  • Discord account — free, create one at discord.com
  • Midjourney subscription — starts at $10/mo (Basic) for ~200 generations/month
  • Join the Midjourney Discord — discord.gg/midjourney

Once joined, find any #newbies-* channel or use the Midjourney bot in your own server. Type /imagine followed by your prompt.

💡 Pro tip: Create your own Discord server and invite the Midjourney bot. This keeps your generations organized and searchable.

The four buttons under every generation:

  • U1-U4 — Upscale one of the four images (higher resolution + details)
  • V1-V4 — Create variations of one image
  • 🔄 (reroll) — Generate new interpretations of the same prompt
  • Web — Open in Midjourney web gallery

2. The Art of Prompting

A great Midjourney prompt has three parts:

ComponentExamplePurpose
Subjecta young fox wearing a steampunk top hatWhat you want to see
Environmentin a Victorian library, warm lightingSetting and atmosphere
Style + Technicalphotorealistic, depth of field, 8k --ar 16:9 --v 6.2Look and format

Golden formula: [Subject] + [Action/Details] + [Environment] + [Lighting] + [Style] + [Technical Parameters]

Examples that actually work:

  • ❌ "beautiful mountain" → too vague, unpredictable results
  • ✅ a majestic mountain peak at sunrise, layers of mist in the valley, alpine flowers in foreground, dramatic golden hour lighting, National Geographic photography, hyperdetailed --ar 16:9 --v 6.2

3. Key Parameters (Your Control Panel)

Parameters are modifiers added with -- at the end of your prompt.

ParameterWhat It DoesExample
--arAspect ratio (width:height)--ar 16:9 for widescreen, --ar 9:16 for portrait
--sStylization (0-1000, default 100)--s 50 more literal, --s 600 more artistic
--cChaos (0-100) — how wild the variations are--c 80 for unexpected surprises
--noNegative — things to exclude--no text, watermark, blur
--iwImage weight (0.5-2.0) for image prompts--iw 1.5 when using a reference image
--vVersion (default 6.2)--v 6.2
🧪 Experiment rule: Change one parameter at a time. Change two and you won't know what caused the difference.

4. Style References (--sref) — Your Cheat Code

Style references are the single most powerful feature in Midjourney v6+. You provide an image that represents the style you want, and Midjourney applies it to your subject.

How to use: Upload a style image to Discord, copy its URL, and add --sref [URL] to your prompt.

Style weight: Control how strongly the style applies with --sw 0-1000 (default 100).

Multi-style mixing: Use multiple --sref URLs separated by spaces.

  • --sref [URL] — Apply style from reference
  • --sw 200 — Stronger style influence
  • --sref [url1] [url2] — Blend multiple styles

Where to find style references: Midjourney's community showcase (midjourney.com/showcase), dedicated style reference channels on Discord, or simply images you like on Pinterest.

5. Consistent Characters with --cref

One of the biggest asks in AI image generation — "how do I get the same character in different scenes?" — is now solved with --cref (character reference).

How it works: Provide a reference image of your character, and Midjourney will maintain their appearance across different prompts.

  • /imagine a knight in armor --cref [character-image-url]
  • /imagine the same character relaxing on a beach --cref [same-url]
  • --cw 0-100 controls character weight (100 = strongest match)

Best practices for character consistency:

  • Use a front-facing, well-lit reference image
  • Avoid busy backgrounds in the reference
  • Keep --cw around 80-100 for faces, lower for poses
  • Use the same --sref for visual consistency across scenes

6. Advanced Techniques

Multi-Prompts with ::

Use :: to assign different weights to parts of your prompt:

a futuristic city::2 cyberpunk style::1.5 rain and neon lights::1 --ar 16:9

This tells Midjourney to prioritize "futuristic city" twice as much as "cyberpunk style."

Image Prompting

Start with an image instead of text. Upload an image, use its URL, and add text to guide it:

[image-url] a watercolor painting of this scene, soft colors --iw 1.5

Remix Mode

Enable /prefer remix to change prompts after a generation while keeping the composition. Perfect for iterating on a layout you like.

7. A Practical Workflow

Here's how professionals use Midjourney in production:

  1. Concept exploration — Use broad prompts with high --c 60-80 for diverse ideas
  2. Pick winners — Select promising compositions, upscale them
  3. Variation refinement — Vary the upscaled image, lower chaos to --c 10-20
  4. Style lock-in — Apply --sref from the best result to maintain consistency
  5. Final upscale — Use /upscale 2x or external tools like Magnific or Topaz Gigapixel

8. Pro Tips from Heavy Users

  • Keep a prompt library. Save winning prompts in a Notion doc or spreadsheet. You'll reuse and remix them.
  • Negative space matters. Add --no text as a default — Midjourney loves writing gibberish.
  • Seed for reproducibility. Find the seed of an image you like (envelope emoji reaction) and use --seed [number] to recreate similar results.
  • Speed vs quality. Fast mode is instant but lower quality. Relax mode is slower (wait in queue) but better results and unlimited for Pro subscribers.
  • Don't fight the aspect ratio. If your subject looks distorted, the --ar might be too extreme. Stick to --ar 4:3 or --ar 3:2 for reliable results.
  • Use Midjourney's web editor. The web interface at midjourney.com has inpainting, outpainting, and more advanced tools than Discord.
